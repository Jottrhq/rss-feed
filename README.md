# RSS Feed Reader

Jottr plugin: `rss-feed`.

This repository is released independently from the main Jottr app. The generated zip and `.sha256` asset are uploaded by Release Please and referenced from the central `Jottrhq/plugins` index.

## Package locally

```bash
python scripts/package-plugin.py --name rss-feed --version 0.1.0
```

## Validate

```bash
python scripts/validate-plugin-json.py plugin.json
```
